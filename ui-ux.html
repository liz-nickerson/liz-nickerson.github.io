U-Go

For my capstone project at UMass, I wrote a proposal for developing a new mobile application, called U-Go. U-Go is used to track Pioneer Valley Transit Authority (PVTA) buses, and is intended as a redesign of the widely used UMass BusTrack.

The proposal contains a set of UI mockups that I created in Axure, as well as a functional specification, a development timeline, and a volume of user research. While developing this project, I worked with UMass BusTrack's owner and solo developer to plan the app's functionality, through user stories.
