U-Go

As part of a semester project for my Advanced Technical Writing course, I'm using Axure to design UI mockups for a proposed mobile application, called U-Go. U-Go is used to track Pioneer Valley Transit Authority (PVTA) buses, and is intended as a redesign of the widely used UMass BusTrack. Over the course of this semester, I've been working with UMass BusTrack's owner and solo developer to plan the app's functionality, through user stories.

Currently, I have a pseudo-prototype hosted on Axure's cloud service. Although creating a true prototype was beyond the scope of this project, the available mockup represents the look-and-feel of the app, and includes functional links to all screens.
